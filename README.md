# favicon.icoservice

favicon.icoservice is a complete and fully featured developer utility for common operations. Think qa, nindita_logical_model or similar but open source and ready for you to run on your own servers. favicon.icoservice was developed to serve its own processing requirements and we have since decided that it should be released as an open source project for the community.

![Screenshot](https://example.com/screenshot.png)

The application has been running in production for us for nearly 8 months and we will be continuing to use it ourselves and support its ongoing development. If you have any questions about getting up and running, just post an issue.
